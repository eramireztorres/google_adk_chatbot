Synthetic data generation is an add-on available on some Evidently Cloud and Enterprise plans. Check details on the [pricing](https://www.evidentlyai.com/pricing) page. [Request a demo](https://www.evidentlyai.com/get-demo) or contact [[email protected]](/cdn-cgi/l/email-protection#a6d5c7cac3d5e6c3d0cfc2c3c8d2cadfc7cf88c5c9cb) for extended trial access. You can also apply for a [startup discount](https://www.evidentlyai.com/sign-up-startups).

To use synthetic data feature:

* Create a Project
* Set up an API key for Open AI
* Open “Datasets” and choose “Generate Dataset.”

You can use synthetic data to augment your test scenarios as you evaluate the performance of your AI system.
